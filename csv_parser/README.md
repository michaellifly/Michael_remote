csvparser
=========

Parses CSV file. Checks valid headers and data using regex and test cases.
run : 
	python main.py --csvpath bs.csv
	

Output:

+-------------------------------------+
| Company Name | Year | Month | Value |
+-------------------------------------+
| Company A    | 2000 | Jan   | 93    |
| Company B    | 2000 | Aug   | 84    |
| Company C    | 1992 | Jan   | 93    |
+-------------------------------------+
